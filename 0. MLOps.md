# MLOps: Continuous Integration (CI), Continuous Delivery (CD) & Continuous Testing (CT)

Motivation: Build a ML model that continuously operate in production

## DevOps vs MLOps
DevOps: Continuous Integration (CI) & Continuous Delivery (CD)

MLOps: Continuous Integration (CI), Continuous Delivery (CD) & Continuous Testing (CT)
